C\+\_\+N, the cyclic group with N elements. The conserved quantities associated with discrete rotationnal symmetry are element of this familly of groups. 

Oftentime called Z\+\_\+N in the literature. This would create a name clash with the infinite group \mbox{\hyperlink{classquantit_1_1conserved_1_1Z}{Z}}.

Note that the implementation limits the length of the cycle to less than 2$^\wedge$16-\/1
