\doxysubsection{Detailed Description}
Pure virtual base class for logging the state of dmrg. This logger is meant to do macro scale logging\+: as such it isn\textquotesingle{}t perfomance critical and we can afford the virtual calls. dmrg call init before the first sweep, it\+\_\+log\+\_\+all at every iterations and call end\+\_\+log\+\_\+all once when the state is converged. the last call to it\+\_\+log\+\_\+all receive the same arguments as end\+\_\+log\+\_\+all (it\textquotesingle{}s not useful to have both function do something if it\+\_\+log\+\_\+all does something at every iterations).
