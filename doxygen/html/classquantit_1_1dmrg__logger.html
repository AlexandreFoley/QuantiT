Detailed Description

Pure virtual base class for logging the state of dmrg. This logger is meant to do macro scale logging: as such it isn't perfomance critical and we can afford the virtual calls. dmrg call init before the first sweep, it_log_all at every iterations and call end_log_all once when the state is converged. the last call to it_log_all receive the same arguments as end_log_all (it's not useful to have both function do something if it_log_all does something at every iterations).
