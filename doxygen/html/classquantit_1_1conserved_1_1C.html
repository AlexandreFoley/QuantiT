Detailed Description

template<uint16_t mod>
class quantit::conserved::C< mod >

C_N, the cyclic group with N elements. The conserved quantities associated with discrete rotationnal symmetry are element of this familly of groups.

Oftentime called Z_N in the literature. This would create a name clash with the infinite group Z.

Note that the implementation limits the length of the cycle to less than 2^16-1
